# OKF Integration for GitHub Copilot

Integrate **Open Knowledge Format (OKF)** with **GitHub Copilot** by creating `.github/copilot-instructions.md`:

```markdown
# GitHub Copilot System Instructions

- Read `.okf/index.md` for overall architecture and domain concepts.
- Follow coding standards documented under `.okf/rules/`.
- Ensure new software components align with service dependencies in `.okf/services/`.
```
